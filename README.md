 # Data Analytics Internship – DecodeLabs

## Project 1: Data Cleaning & Preparation

In this project, I cleaned an e-commerce order dataset (1,200 records) using Python (pandas) in Google Colab. I handled missing values, checked for duplicates, standardized date formats (YYYY-MM-DD), and fixed floating-point rounding errors in the pricing columns.

**Tools used:** Python, pandas, Google Colab, Excel (openpyxl)

**Results:**
- Filled 309 missing values in the CouponCode column
- 0 duplicate records found
- Standardized date and numeric formats across the entire dataset
